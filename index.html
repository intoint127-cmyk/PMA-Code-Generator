↑ อัพเดท Database
เลือกวิธีที่ต้องการอัพเดทไฟล์ PMA_2025 ด้วยรหัสใหม่ที่สร้างขึ้น
✅ ดาวน์โหลดไฟล์ PMA_2025 ที่อัพเดทแล้วได้ทันที (รหัสใหม่ถูกเพิ่มเข้า Sheet ที่ถูกต้องแล้ว)
• ไฟล์ที่ดาวน์โหลดจะมี Sheet ครบทั้งหมด
• รหัสใหม่จะถูกเพิ่มต่อท้ายในแต่ละ Sheet
• สามารถนำไปอัพโหลดแทนที่ไฟล์เดิมได้เลย
⚠️ การเขียนไปยัง Google Sheets ต้องการ Service Account JSON Key จาก Google Cloud Console
วิธีได้ Key: Google Cloud Console → IAM & Admin → Service Accounts → Create → Download JSON
จากนั้น Share Spreadsheet ให้ service account email ใน Key มีสิทธิ์ Editor
หรือ: ดาวน์โหลด Excel แล้วอัพโหลดแทนที่ใน Google Sheets
ℹ️ วิธีง่ายกว่า: ใช้แท็บ "ดาวน์โหลด Excel" แล้วนำไฟล์ไป Import ทับใน Google Sheets
⚠️ การเขียนไปยัง SharePoint ต้องการ Azure App Registration (Client ID + Secret)
ℹ️ วิธีง่ายกว่า: ใช้แท็บ "ดาวน์โหลด Excel" แล้วนำไฟล์ไป Upload แทนที่ใน SharePoint
